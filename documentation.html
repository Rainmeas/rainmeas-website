Getting Started

Rainmeas is a powerful package manager for Rainmeter skins that simplifies the installation, updating, and management of skin modules. Whether you're a beginner or an experienced Rainmeter user, Rainmeas makes it easy to enhance your desktop with beautiful skins.

Before getting started, ensure you have Rainmeter installed on your system. You can download Rainmeter from the official website.

Installation

Using the Installer (Recommended)

Download the latest installer (rainmeas-setup.exe) from the releases page and run it to install Rainmeas with automatic PATH integration.

Manual Installation

To install Rainmeas manually, run:

pip install .

Or for development:

pip install -e .

Basic Usage

Initializing Rainmeas

Initialize Rainmeas in a Rainmeter skin directory:

rainmeas init

Installing Packages

Install a package:

rainmeas install freshweather

Install a specific version of a package:

rainmeas install freshweather@1.3.0

Managing Packages

Listing Installed Packages

rainmeas list

Searching for Packages

rainmeas search weather

Updating Packages

Update all packages:

rainmeas update

Update a specific package:

rainmeas update freshweather

Removing Packages

rainmeas remove freshweather

Configuration

Configuration File

Rainmeas uses a configuration file located at .rainmeas/config.json in your Rainmeter skin directory. This file stores settings and package metadata.

Environment Variables

Rainmeas respects the following environment variables:

  • RAINMEAS_REGISTRY_URL - Custom registry URL
  • RAINMEAS_CACHE_DIR - Custom cache directory
  • RAINMEAS_LOG_LEVEL - Logging level (debug, info, warn, error)

Troubleshooting

Common Issues

Package installation fails:

  • Ensure you have internet connectivity
  • Check that the package name is correct
  • Verify you have write permissions to the Rainmeter skins directory

Rainmeas command not found:

  • If installed via installer, restart your terminal/command prompt
  • If installed manually, ensure Python scripts directory is in your PATH

Verifying Package Integrity

You can verify package integrity with:

rainmeas verify

Contributing

Publishing Packages

To publish your own Rainmeas packages:

  1. Create a package manifest file (package.json)
  2. Host your package on a public repository
  3. Submit your package to the Rainmeas registry

Developing Rainmeas

To contribute to Rainmeas development:

  1. Fork the Rainmeas repository
  2. Create a feature branch
  3. Implement your changes
  4. Submit a pull request
