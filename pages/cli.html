init

rainmeas init

Initialize Rainmeas in a Rainmeter skin directory. This creates the necessary configuration files and directories.

Usage

rainmeas init

Example

$ cd C:\Users\YourName\Documents\Rainmeter\Skins\MySkin
$ rainmeas init
Rainmeas initialized successfully

install

rainmeas install [<package>[@<version>]]

Install a package or a specific version of a package. If no package is specified, installs all packages listed in rainmeas-package.json in the current directory. If no version is specified, the latest version is installed.

Usage

rainmeas install
rainmeas install <package>
rainmeas install <package>@<version>

Options

  • --force - Force installation even if package is already installed
  • --save - Save package to dependencies (default behavior)
  • --no-save - Install without saving to dependencies

Examples

# Install all packages from rainmeas-package.json
$ rainmeas install
Installing packages from rainmeas-package.json...
✓ Successfully installed nurashadeweather@1.1.0
✓ Successfully installed nurashade-reversegeo@1.0.0

# Install the latest version of a specific package
$ rainmeas install nurashadeweather
Installing nurashadeweather...
✓ Successfully installed nurashadeweather@1.1.0

# Install a specific version
$ rainmeas install nurashadeweather@1.0.0
Installing nurashadeweather@1.0.0...
✓ Successfully installed nurashadeweather@1.0.0

remove

rainmeas remove <package>

Remove an installed package and clean up associated files.

Usage

rainmeas remove <package>

Options

  • --force - Force removal without confirmation

Example

$ rainmeas remove nurashadeweather
Removing nurashadeweather...
✓ Successfully removed nurashadeweather

update

rainmeas update [<package>]

Update all packages or a specific package to the latest version.

Usage

rainmeas update
rainmeas update <package>

Examples

# Update all packages
$ rainmeas update
Updating packages...
✓ Updated nurashadeweather from 1.0.0 to 1.1.0
✓ All packages are up to date

# Update a specific package
$ rainmeas update nurashadeweather
Updating nurashadeweather...
✓ Updated nurashadeweather from 1.0.0 to 1.1.0

list

rainmeas list

List all installed packages and their versions.

Usage

rainmeas list

Options

  • --json - Output in JSON format
  • --depth <level> - Limit dependency tree depth

Example

$ rainmeas list
Installed packages:
nurashadeweather@1.1.0
musicvisualizer@2.1.4
systemmonitor@3.0.1

search

rainmeas search <query>

Search for packages in the registry that match the query.

Usage

rainmeas search <query>

Options

  • --limit <number> - Limit number of results (default: 20)
  • --sort <field> - Sort by field (name, downloads, updated)

Example

$ rainmeas search weather
Found 5 packages:
nurashadeweather - Beautiful weather skin with multiple themes
weatherplus - Advanced weather with forecast and alerts
simpleweather - Minimalist weather display
stormtracker - Severe weather alerts and radar
climateview - Climate data and historical weather

verify

rainmeas verify

Verify the integrity of installed packages and configuration files.

Usage

rainmeas verify

Example

$ rainmeas verify
Verifying package integrity...
✓ nurashadeweather@1.0.0 - OK
✓ nurashade-reversegeo@1.0.0 - OK

Verification summary: 2 verified, 0 missing

clean

rainmeas clean

Remove unused modules and clean up the cache.

Usage

rainmeas clean

Example

$ rainmeas clean
Removed unused module: nurashade-reversegeo
Cleaned 1 unused modules

version

rainmeas version

Display the current version of Rainmeas CLI.

Usage

rainmeas version

Example

$ rainmeas version
rainmeas 0.0.1

help

rainmeas help [<command>]

Display help information for all commands or a specific command.

Usage

rainmeas help
rainmeas help <command>

Example

$ rainmeas help
Usage: rainmeas [command] [options]

Commands:
  init       Initialize Rainmeas in a directory
  install    Install a package
  remove     Remove a package
  update     Update packages
  list       List installed packages
  search     Search for packages
  info       Show package information
  verify     Verify package integrity
  clean      Clean unused modules
  version    Show CLI version
  help       Show help

Run 'rainmeas help <command>' for more information on a specific command.
