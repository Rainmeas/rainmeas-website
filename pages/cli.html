init

rainmeas init

Initialize Rainmeas in a Rainmeter skin directory. This creates the necessary configuration files and directories.

Usage

rainmeas init

Example

$ cd C:\Users\YourName\Documents\Rainmeter\Skins\MySkin
$ rainmeas init
Initialized Rainmeas in C:\Users\YourName\Documents\Rainmeter\Skins\MySkin

install

rainmeas install <package>[@<version>]

Install a package or a specific version of a package. If no version is specified, the latest version is installed.

Usage

rainmeas install <package>
rainmeas install <package>@<version>

Options

  • --force - Force installation even if package is already installed
  • --save - Save package to dependencies (default behavior)
  • --no-save - Install without saving to dependencies

Examples

# Install the latest version
$ rainmeas install freshweather
Installing freshweather...
✓ Successfully installed freshweather@1.3.0

# Install a specific version
$ rainmeas install freshweather@1.2.0
Installing freshweather@1.2.0...
✓ Successfully installed freshweather@1.2.0

remove

rainmeas remove <package>

Remove an installed package and clean up associated files.

Usage

rainmeas remove <package>

Options

  • --force - Force removal without confirmation

Example

$ rainmeas remove freshweather
Removing freshweather...
✓ Successfully removed freshweather

update

rainmeas update [<package>]

Update all packages or a specific package to the latest version.

Usage

rainmeas update
rainmeas update <package>

Options

  • --check - Check for updates without installing
  • --force - Force update even if already up-to-date

Examples

# Update all packages
$ rainmeas update
Updating packages...
✓ Updated freshweather from 1.2.0 to 1.3.0
✓ All packages are up to date

# Update a specific package
$ rainmeas update freshweather
Updating freshweather...
✓ Updated freshweather from 1.2.0 to 1.3.0

list

rainmeas list

List all installed packages and their versions.

Usage

rainmeas list

Options

  • --json - Output in JSON format
  • --depth <level> - Limit dependency tree depth

Example

$ rainmeas list
Installed packages:
freshweather@1.3.0
musicvisualizer@2.1.4
systemmonitor@3.0.1

search

rainmeas search <query>

Search for packages in the registry that match the query.

Usage

rainmeas search <query>

Options

  • --limit <number> - Limit number of results (default: 20)
  • --sort <field> - Sort by field (name, downloads, updated)

Example

$ rainmeas search weather
Found 5 packages:
freshweather - Beautiful weather skin with multiple themes
weatherplus - Advanced weather with forecast and alerts
simpleweather - Minimalist weather display
stormtracker - Severe weather alerts and radar
climateview - Climate data and historical weather

info

rainmeas info <package>

Display detailed information about a package.

Usage

rainmeas info <package>

Example

$ rainmeas info freshweather
Name: freshweather
Version: 1.3.0
Author: Someone123
Description: A beautiful and customizable weather skin for Rainmeter with multiple themes and layouts.
Homepage: https://github.com/someone123/freshweather
License: MIT
Downloads: 12,500
Dependencies: none

verify

rainmeas verify

Verify the integrity of installed packages and configuration files.

Usage

rainmeas verify

Example

$ rainmeas verify
Verifying packages...
✓ freshweather: OK
✓ musicvisualizer: OK
✓ systemmonitor: OK
All packages verified successfully

clean

rainmeas clean

Remove unused modules and clean up the cache.

Usage

rainmeas clean

Options

  • --cache - Clean only the cache directory
  • --modules - Clean only unused modules
  • --force - Force cleaning without confirmation

Example

$ rainmeas clean
Cleaning unused modules...
Removed 3 unused modules
Cache cleaned successfully
Freed 15.2 MB of disk space

version

rainmeas version

Display the current version of Rainmeas CLI.

Usage

rainmeas version

Example

$ rainmeas version
Rainmeas CLI v1.2.3

help

rainmeas help [<command>]

Display help information for all commands or a specific command.

Usage

rainmeas help
rainmeas help <command>

Example

$ rainmeas help
Usage: rainmeas [command] [options]

Commands:
  init       Initialize Rainmeas in a directory
  install    Install a package
  remove     Remove a package
  update     Update packages
  list       List installed packages
  search     Search for packages
  info       Show package information
  verify     Verify package integrity
  clean      Clean unused modules
  version    Show CLI version
  help       Show help

Run 'rainmeas help ' for more information on a specific command.
