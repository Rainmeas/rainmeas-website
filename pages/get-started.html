Get Started

Learn how to install and set up Rainmeas to begin managing your Rainmeter modules.

Prerequisites

Before installing Rainmeas, ensure you have the following:

  • Rainmeter - Rainmeas works with Rainmeter 4.3 or later. Download it from the official website.
  • Windows - Rainmeas is designed for Windows operating systems.
  • Internet Connection - Required for downloading packages from the registry.

Installation

Using the Installer (Recommended)

Download the latest installer (rainmeas-setup.exe) from the releases page and run it to install Rainmeas with automatic PATH integration.

Manual Installation

If you prefer manual installation:

  1. Ensure you have Python 3.7 or later installed on your system
  2. Clone the Rainmeas repository or download the source code
  3. Navigate to the project directory and install dependencies: 
    pip install -r requirements.txt
  4. Run the setup script: 
    python setup.py install

Initialization

After installing Rainmeas, initialize it in your Rainmeter modules directory: 

rainmeas init

This command creates the necessary configuration files and directories for managing your packages.

Installing Your First Package

Once initialized, you can install packages from the registry: 

rainmeas install nurashadeweather

This will download and install the "nurashadeweather" package to your Rainmeter modules directory.
