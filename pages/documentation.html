Getting Started

Rainmeas is a powerful package manager for Rainmeter modules that simplifies the installation, updating, and management of reusable components. Whether you're a beginner or an experienced Rainmeter user, Rainmeas makes it easy to enhance your desktop with modular components.

Before getting started, ensure you have Rainmeter installed on your system. You can download Rainmeter from the official website.

Installation

Using the Installer (Recommended)

Download the latest installer (rainmeas-setup.exe) from the releases page and run it to install Rainmeas with automatic PATH integration.

Basic Usage

Initializing Rainmeas

Initialize Rainmeas in a Rainmeter modules directory:

rainmeas init

Installing Packages

Install a package:

rainmeas install nurashadeweather

Install a specific version of a package:

rainmeas install nurashadeweather@1.1.0

Managing Packages

Listing Installed Packages

rainmeas list

Searching for Packages

rainmeas search weather

Updating Packages

Update all packages:

rainmeas update

Update a specific package:

rainmeas update nurashadeweather

Removing Packages

rainmeas remove nurashadeweather

Troubleshooting

Common Issues

Package installation fails:

  • Ensure you have internet connectivity
  • Check that the package name is correct
  • Verify you have write permissions to the Rainmeter modules directory

Rainmeas command not found:

  • If installed via installer, restart your terminal/command prompt
  • If installed manually, ensure Python scripts directory is in your PATH

Verifying Package Integrity

You can verify package integrity with:

rainmeas verify

Contributing

Publishing Packages

To publish your own Rainmeas packages:

  1. Create a package manifest file (package.json)
  2. Host your package on a public repository
  3. Submit your package to the Rainmeas registry

Developing Rainmeas

To contribute to Rainmeas development:

  1. Fork the Rainmeas repository
  2. Create a feature branch
  3. Implement your changes
  4. Submit a pull request

CLI Commands

Complete reference for all Rainmeas command-line interface commands and options.

init

rainmeas init

Initialize Rainmeas in a Rainmeter modules directory. This creates the necessary configuration files and directories.

Usage

rainmeas init

install

rainmeas install [<package>[@<version>]]

Install a package or a specific version of a package. If no package is specified, installs all packages listed in rainmeas-package.json in the current directory. If no version is specified, the latest version is installed.

Usage

rainmeas install
rainmeas install <package>
rainmeas install <package>@<version>

remove

rainmeas remove <package>

Remove an installed package and clean up associated files.

Usage

rainmeas remove <package>

update

rainmeas update [<package>]

Update all packages or a specific package to the latest version.

Usage

rainmeas update
rainmeas update <package>

list

rainmeas list

List all installed packages and their versions.

Usage

rainmeas list

search

rainmeas search <query>

Search for packages in the registry that match the query.

Usage

rainmeas search <query>

info

rainmeas info <package>

Display detailed information about a package.

Usage

rainmeas info <package>

verify

rainmeas verify

Verify the integrity of installed packages and configuration files.

Usage

rainmeas verify

clean

rainmeas clean

Remove unused modules and clean up the cache.

Usage

rainmeas clean

version

rainmeas version

Display the current version of Rainmeas CLI.

Usage

rainmeas version

help

rainmeas help [<command>]

Display help information for all commands or a specific command.

Usage

rainmeas help
rainmeas help <command>
